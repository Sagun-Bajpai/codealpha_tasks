# Music Player

This project is a Music Player built using HTML, CSS, and JavaScript as part of the CodeAlpha Frontend Development Internship.

## Features

- Play and Pause functionality
- Next and Previous track controls
- Display of song title and artist
- Track duration display
- Progress bar for song playback
- Volume control
- Playlist functionality
- Autoplay feature

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Description

The Music Player provides an interactive interface where users can play and control music tracks. Users can navigate between songs using next and previous buttons, control the volume, and track playback progress using the progress bar. The interface is designed to be visually appealing and responsive.

## Internship Task

This project was developed as **Task 4** for the CodeAlpha Frontend Development Internship.

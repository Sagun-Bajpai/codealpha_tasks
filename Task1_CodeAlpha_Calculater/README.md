# Image Gallery

This project is an Image Gallery created as part of the CodeAlpha Frontend Development Internship.

The gallery is designed using HTML and CSS for layout and JavaScript for interactive functionality.

## Features

- Responsive image gallery layout
- Next and Previous navigation buttons
- Lightbox view for displaying images
- Hover effects on images
- Smooth transitions for better user experience
- Responsive design for different screen sizes

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Description

The Image Gallery allows users to browse through a collection of images using navigation controls. When an image is clicked, it opens in a lightbox view where users can easily view the image in a larger format. The layout is responsive and adjusts according to different screen sizes.

## Internship Task

This project was developed as **Task 1** for the CodeAlpha Frontend Development Internship.
